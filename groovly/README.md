# 🕺 GROOVLY — Learn. Dance. Earn.

A platform for dancers: live classes, video courses, teacher earnings, and videographer booking.

## Run it

```bash
npm install
npm run dev      # local dev server → http://localhost:5173
npm run build    # production build → dist/
```

## Features (current)

- **Home** — Promo carousel (paid promotions), Live Classes, Video library
- **Learn** — Student learning hub
- **Earn** — Teacher Studio: earnings dashboard (70% revenue share), video upload via front/back camera recording or file upload, publish flow
- **Book Pro** — Book vetted dance videographers by city, with portfolios & ratings
- **Groove AI** — Customer support chat
- **Currency** — Toggle INR ⇄ USD everywhere
- Fully responsive: desktop nav + mobile bottom-tab navigation

## Stack

Vite · React 18 · TypeScript · Tailwind CSS v4 · lucide-react
